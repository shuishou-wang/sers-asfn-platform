# SERS-ASFN Diabetes Auxiliary Detection Platform

This repository contains the public static web version of the SERS-ASFN auxiliary detection platform.

## Deployment

This is a static website and can be published directly with GitHub Pages.

GitHub Pages settings:

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

## Files

- index.html: entry page
- styles.css: interface styles
- app.js: interaction logic
- platform-data.json: display data

## Boundary

This system is for research assistance and interface demonstration only. It does not replace clinical diagnosis.
